※ 本アプリはローカル環境で起動する必要があります。

※ ターミナル（黒い画面）を使用します。

※ Node.js がインストールされている前提です。


## 動作確認方法（履修登録システム）

### 1. バックエンド（画面の裏側の処理）を起動します

1. ダウンロードしたフォルダを開き、ターミナルを起動します。
2. 以下のコマンドを上から順に入力してください。
   
cd BackEnd

cd project

node index.mjs

3. エラーが表示されず、サーバーが起動すれば成功です。

---

### 2. フロントエンド（画面）を起動します

1. 別のターミナルを開き、以下のコマンドを入力してください。

cd FrontEnd

npm run dev

2. ターミナルに表示される  
   http://localhost:xxxx`
   をブラウザで開いてください。

---

### 3. ログイン方法

・学籍番号：student@example.com
・パスワード：password

ログイン後、履修登録・時間割・検索機能を確認できます。
